Para el correcto funcionamiento del proyecto Laravel se deben de ejecutar estos dos comandos abriendo una consola en el directorio raíz del proyecto:
- ```composer install```
-  ```npm install```
<br>
Posteriormente para activar el servidor de Laravel habrá que introducir el comando:
<br>
-  ```php artisan serve```
<br>
Además las consultas de Postman están configuradas de manera que el puerto del servidor Laravel deberá de ser el 8000, mientras que el del proyecto Java/Spring Boot será el 8080.
